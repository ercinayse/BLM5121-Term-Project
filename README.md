# GündemRadar - Hibrit Haber Arama Sistemi

Bu proje, BLM5121 Web Madenciliği dersi kapsamında geliştirilmiştir.

Projede bilgi erişim (Information Retrieval), web mining ve graph tabanlı sıralama yöntemleri kullanılarak Türkçe haber içerikleri üzerinde çalışan bir haber arama sistemi geliştirilmiştir.

Sistem içerisinde:

- BM25
- PageRank
- HITS

algoritmaları birlikte kullanılmıştır.

Projede ayrıca:

- veri toplama,
- preprocessing,
- inverted index oluşturma,
- graph analizi,
- hibrit sıralama,
- REST API geliştirme,
- frontend tasarımı

gibi web madenciliği ve bilgi erişim sistemlerine ait temel bileşenler uygulanmıştır.

## Kullanılan Teknolojiler

- Python
- Flask
- PostgreSQL
- Next.js
- BeautifulSoup
- NetworkX
